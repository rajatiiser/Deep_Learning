# Text Classification using Simple RNN and LSTM

## 📌 Project Overview

This project compares the performance of Simple Recurrent Neural Networks (Simple RNN) and Long Short-Term Memory (LSTM) networks for text classification using the 20 Newsgroups dataset.

The objective is to analyze how both architectures learn sequential text data and evaluate their effectiveness in multi-class document classification.

---

## 📂 Dataset Information

* Dataset: 20 Newsgroups
* Task: Multi-Class Text Classification
* Number of Categories: Multiple news topics

The dataset contains news articles grouped into different categories, making it suitable for sequence learning and text classification tasks.

---

## 🔄 Data Preprocessing

The following preprocessing steps were performed:

* Text Cleaning
* Tokenization
* Sequence Padding
* One-Hot Encoding

These steps convert text documents into numerical sequences suitable for deep learning models.

---

## 🧠 Models Implemented

### Simple RNN

A basic recurrent neural network capable of processing sequential data.

### LSTM

An advanced recurrent architecture designed to capture long-term dependencies and reduce the vanishing gradient problem.

---

## 📊 Evaluation Metrics

* Accuracy
* Loss Curves

---

## 📈 Results

Both models were evaluated on the same dataset. The comparison highlights the strengths and limitations of each architecture when handling sequential text data.

---

## 📌 Conclusion

This project demonstrates the application of recurrent neural networks to text classification. The results provide insights into the differences between Simple RNN and LSTM models and show how LSTM can better capture contextual information in sequential data.
